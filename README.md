# Capstone 3: Analysis with Power BI

## Project Overview
This project analyzes sales performance for EmporiUm's South Region, including Florida, South Carolina, and Texas, from 2022 to 2025 using Power BI. The purpose of this report is to identify sales trends, compare state performance, analyze product categories, and provide insights into general book sales and author performance.

---

## Data Sources
The report was created using the following datasets:

- Store Sales data
- Products table
- Store Locations table
- Management Team table
- Book List text file containing general book titles and authors

---

## Report Pages

### 1. Sales Overview
This page provides an executive dashboard of overall sales performance in the South Region and includes:

- Total South Region Revenue
- Total Transactions
- Average Sale Amount
- Revenue Trend (2022–2025)
- Revenue by State
- Revenue by Product Category
- Revenue by Store Location
- Revenue by Store Manager and State

### 2. Regional Book Insights
This page focuses on general book performance and includes:

- Top 10 Best-Selling Books and their authors
- General Books Revenue card
- Top Authors by Revenue
- Fiction vs. Non-Fiction Revenue comparison
- Average Revenue Per Book by Author (Treemap)

### 3. State Drill-Through
This page was created as an interactive bonus feature and allows users to drill through from the main dashboard to view state-specific metrics and insights.

---

## Key Insights

- Florida generated the highest revenue among the three states.
- Technology & Accessories was the highest-performing product category.
- General book sales generated approximately **$138K** in revenue.
- Fiction titles slightly outperformed Non-Fiction titles in total revenue.
- Several authors demonstrated high average revenue per book, indicating strong performance on an individual title basis.
