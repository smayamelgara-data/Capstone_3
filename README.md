# Capstone 3: Analysis with Power BI

This project analyzes EmporiUm's South Region (Florida, South Carolina, and Texas) sales data from 2022–2025 using Power BI.

## Report Pages
- Sales Overview dashboard with revenue trends, state performance, product categories, and key metrics.
- Regional Book Insights dashboard featuring top-selling books, author performance, and fiction vs. non-fiction revenue.

## Bonus Features
- State drill-through page
- Mobile report layout
- Chart showing Average Revenue Per Book by Author

## Key Findings
- Florida generated the highest revenue in the South Region.
- Technology & Accessories was the top-performing product category.
- General books generated approximately $138K in revenue, with Fiction slightly outperforming Non-Fiction.

## Project Outcome
This Power BI report transforms raw sales and book data into an interactive dashboard that helps management quickly understand regional performance, identify top-performing products and authors, and make more informed business decisions.
